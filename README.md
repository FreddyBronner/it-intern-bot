IT-Intern Bot — Telegram-бот-агрегатор IT-стажировок

Telegram-бот для поиска IT-стажировок в крупных компаниях РФ.
Часть проекта IT-Intern — единого сервиса подачи заявок на стажировки.

Возможности

1) Поиск по стеку - Python, Go, Java, C++ и др.
2) Фильтр по компаниям - Яндекс, Сбер, VK, Т-Банк, Ozon и др.
3) Фильтр по городу** - Москва, Санкт-Петербург, Екатеринбург + удалёнка
4) Избранное - сохраняйте интересные вакансии
5) Подписки - уведомления о новых стажировках по вашему стеку
6) Профиль - укажите стек, город и курс для персонализации

Технологии

- Python
- aiogram 3.x
- aiosqlite
- SQLite

Алгоритм:

1. Создайте бота в Telegram

Напишите [@BotFather], отправьте `/newbot` и получите токен.

2. Установите зависимости

```bash
pip install -r requirements.txt
```

3. Заполните базу данных

```bash
python seed_data.py
```
4. Запустите бота

Windows (PowerShell)

```bash
$env:BOT_TOKEN="токен"
python bot.py
```



users          - профили пользователей (стек, город, курс)
internships    - вакансии (компания, стек, город, зарплата, дедлайн)
favorites      - избранные вакансии пользователей
subs
criptions  - подписки на уведомления

Команды бота

/start     Приветствие и главное меню
/profile   Настройка профиля
/search    Поиск стажировок 
/help      Справка по командам
